JavaScript strings are primitive values. Also, strings are immutable. It means that if you modify a string, you will always get a new string. The original string doesn't change. To create literal strings, you use either single quotes (') or double quotes (") like this: let str = 'Hi'; let greeting = "Hello";

String Methods

charAt(): Retorna o caractere na posição especificada de uma string.

"hello".charAt(0); // Retorna "h"

charCodeAt(): Retorna o valor Unicode do caractere na posição especificada de uma string.

"hello".charCodeAt(1); // Retorna 101 (valor Unicode de "e")

concat(): Combina o texto de duas ou mais strings e retorna uma nova string.

"Hello,".concat(" world!"); // Retorna "Hello, world!"

includes(): Verifica se uma string contém outra string. Retorna true ou false.

"Hello, world!".includes("world"); // Retorna true

endsWith(): Verifica se uma string termina com outra string. Retorna true ou false.

"Hello, world!".endsWith("world!"); // Retorna true

fromCharCode(): Converte valores Unicode em caracteres.

String.fromCharCode(65); // Retorna "A"

indexOf(): Retorna a posição da primeira ocorrência de uma string dentro de outra string.

"Hello, world!".indexOf("world"); // Retorna 7

lastIndexOf(): Retorna a posição da última ocorrência de uma string dentro de outra string.

"Hello, world, hello!".lastIndexOf("hello"); // Retorna 13

localeCompare(): Compara duas strings em uma ordem de classificação específica.

"a".localeCompare("b"); // Retorna -1

match(): Recupera as correspondências de uma string em relação a uma expressão regular.

"Hello, world!".match(/world/); // Retorna ["world"]

padEnd(): Preenche o final de uma string com um determinado número de caracteres.

"Hello".padEnd(10, " world!"); // Retorna "Hello worl"

padStart(): Preenche o início de uma string com um determinado número de caracteres.

"world!".padStart(10, "Hello, "); // Retorna "Hello, world!"

repeat(): Retorna uma nova string contendo a string original repetida um determinado número de vezes.

"hello ".repeat(3); // Retorna "hello hello hello "

replace(): Substitui parte de uma string por outra string.

"Hello, world!".replace("world", "everyone"); // Retorna "Hello, everyone!"

search(): Pesquisa uma string para encontrar uma correspondência com uma expressão regular e retorna a posição da correspondência.

"Hello, world!".search(/world/); // Retorna 7

slice(): Extrai uma parte de uma string e retorna uma nova string.

"Hello, world!".slice(7, 12); // Retorna "world"

split(): Divide uma string em substrings e retorna um array.

"Hello, world!".split(", "); // Retorna ["Hello", "world!"]

startsWith(): Verifica se uma string começa com outra string. Retorna true ou false.

"Hello, world!".startsWith("Hello"); // Retorna true

substr(): Extrai caracteres de uma string, começando em uma posição especificada e por uma quantidade determinada de caracteres.

"Hello, world!".substr(7, 5); // Retorna "world"

substring(): Extrai os caracteres entre duas posições em uma string.

"Hello, world!".substring(7, 12); // Retorna "world"

toLocaleLowerCase(): Converte todos os caracteres de uma string em letras minúsculas, de acordo com a localidade.

"Hello, WORLD!".toLocaleLowerCase(); // Retorna "hello, world!"

toLocaleUpperCase(): Converte todos os caracteres de uma string em letras maiúsculas, de acordo com a localidade.

"hello, world!".toLocaleUpperCase(); // Retorna "HELLO, WORLD!"

toLowerCase(): Converte todos os caracteres de uma string em letras minúsculas.

"Hello, WORLD!".toLowerCase(); // Retorna "hello, world!"

toString(): Retorna uma string representando o objeto especificado.

var str = new String("Hello, world!"); str.toString(); // Retorna "Hello, world!"

toUpperCase(): Converte todos os caracteres de uma string em letras maiúsculas.

"hello, world!".toUpperCase(); // Retorna "HELLO, WORLD!"

trim(): Remove os espaços em branco do início e do final de uma string.

" Hello, world! ".trim(); // Retorna "Hello, world!"

valueOf(): Retorna o valor primitivo de um objeto String.

var str = new String("Hello, world!"); str.valueOf(); // Retorna "Hello, world!"
